# Obsidian Vault 사용 안내

이 디렉터리를 Obsidian 에서 **Open folder as vault** 로 열어 사용합니다.

## 폴더 구조 (8종 문서유형 기반)
```
00_공통관리/        — 문서체계·번호체계·용어집·문서관리대장(링크)
00_MOC/            — 인덱스(Map of Content)
01_구성원칙/       — 최상위 기준 (표준프로세스_구성원칙.md)
02_표준/           — 표준별 작업공간 (예: ISO9001/)
03_POL_정책/       — POL (정책서)
04_PRO_절차/       — PRO (절차서)
05_WI_업무지침/    — WI  (업무지침서)
06_TMP_템플릿/     — TMP (빈 양식)
07_EX_작성예시/    — EX  (작성예시)
08_REC_기록/       — REC (실제 기록 - 운영단계에서 생성)
09_REF_참고자료/   — REF (외부 규정·가이드 요약)
90_MAT_통합매핑/   — MAT (관리대장 5종 + 표준별 추적성)
99_템플릿/         — Obsidian Templates 플러그인용
99_폐기_보관/      — 만료/폐지 문서
```

## 파일 네이밍 규칙
```
[유형]-[식별번호]_[문서명]_v[버전].md
```

| POL | `POL-QMS-001_품질방침_v1.0.md` |
