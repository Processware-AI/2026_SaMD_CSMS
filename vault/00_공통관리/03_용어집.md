# 전사 용어집

> 모든 산출물에서 사용하는 용어의 단일 출처. 항목 추가 시 알파벳/가나다 순 유지.

| 용어 | 영문 | 정의 | 관련 표준 |
|---|---|---|---|
| CAPA | Corrective Action / Preventive Action | 시정·예방조치 | ISO 9001 |
| HLS | High-Level Structure | ISO 경영시스템 표준의 공통 목차(조항 4~10) | ISO Annex SL |
| IMS | Integrated Management System | 품질·환경·보안 등 여러 경영시스템을 HLS 기반으로 통합 운영 | ISO Annex SL |
| KPI | Key Performance Indicator | 핵심 성과지표 | — |
| MOC | Map of Content | Obsidian 허브 노트(목차·랜딩 페이지) | — |
| PDCA | Plan-Do-Check-Act | 지속 개선 사이클 | ISO 9001 |
| PDD | Process Definition Document | 프로세스 정의서. 본 체계에서는 **PRO** 에 해당 | — |
| RACI | Responsible/Accountable/Consulted/Informed | 책임 분담 매트릭스 | — |
| SBOM | Software Bill of Materials | 소프트웨어 구성 명세 | NIST |
| SEPG | Software Engineering Process Group | 프로세스 엔지니어링·품질보증 조직 | CMMI |
