## 3. 작성 지침

1. 협조 번호는 `MSP-VA-{연도}-{순번}` 형식으로 부여하며, 관련 취약점 통보 번호(MSP-VUL-)를 반드시 연계 기재한다.
2. 제조업자등에 제공하는 로그, 설정 정보 등은 환자 개인정보가 포함되지 않도록 사전 비식별화 처리 후 제공한다.
3. 접근 권한 부여 시 허용 범위와 기간을 명확히 한정한다.
4. 패치 적용 시 임상 업무 영향을 최소화하는 일정을 협의하여 기재한다.
5. 협조 완료 후 결과를 정리하여 내부 보안 기록으로 보관한다.
