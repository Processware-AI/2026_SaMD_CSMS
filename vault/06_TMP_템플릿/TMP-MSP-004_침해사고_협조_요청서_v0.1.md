## 3. 작성 지침

1. 요청 번호는 `MSP-COO-{연도}-{순번}` 형식으로 부여하며, 관련 침해사고 신고 번호(MSP-INC-)를 반드시 연계 기재한다.
2. 접근 권한 부여 시 허용 기간과 범위를 명확히 한정하고, 병원 내부 보안 정책 준수 조건을 명기한다.
3. 현장 방문 지원 시 병원 측 안내 담당자를 사전 지정하여 기재한다.
4. 제조업자등에 제공하는 정보의 비밀유지 조건은 기존 계약 조건을 따르되, 추가 사항이 있으면 비고란에 기재한다.
5. 협조 완료 후 결과를 별도 기록으로 관리하고, 침해사고 사후 보고서에 반영한다.
