# SaMD-CSMS 입력자료

> 상위 규약: [[05_입력자료_규칙]]

## 표준 개요 (입력자료 기반)
- **표준명**: 디지털 의료기기 사이버보안 관리 체계 (SaMD Cybersecurity Management System)
- **발행처**: 식품의약품안전처 / 식품의약품안전평가원 (추정 — 본문 확인 후 갱신)
- **구성**: 본체(관리체계) + 세부지침(22조) + 보조자료 2종 = 25 파일
- **한국어 공공 가이드** (공공저작물 추정 — 라이선스 확인 필요)
