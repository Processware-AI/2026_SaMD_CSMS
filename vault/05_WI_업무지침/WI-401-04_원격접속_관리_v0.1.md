# 원격접속 관리 업무지침 (WI-401-04)

> 상위 절차: [[PRO-MDCS-401_접근통제_및_인증_관리_절차_v1.0]] · §5 단계 6

## 1. 업무 목적

의료기기 운영 환경에 대한 외부 원격 접속을 **기본 차단**하고, 업무상 불가피한 경우에 한해 MFA·VPN/SSH·접근 IP 제한·허용 시간·세션 기록을 갖춘 **제한적 허용** 조건을 정의한다.

## 2. 수행 주체

- **주 수행자**: IAM 관리자, 네트워크팀
- **검토자**: 시스템 소유자(업무 필요성), 오동석(보안 영향)
- **승인자**: 오동석

## 3. 범위

- 임직원의 재택·출장 시 운영 환경 접속
- 외주 개발·유지보수 벤더의 원격 지원
- MSP(병원 현장) 에 설치된 기기에 대한 제조업자 원격 지원
- 클라우드 관리 콘솔·DB 관리 콘솔 등 관리자 원격 접속

## 4. 입력 자료 / 산출물

- **Input**
  - [[TMP-ACC-004_원격접속_신청서_v0.1]] (사유·대상·기간·IP)
  - 계정 인벤토리 ([[WI-401-01_계정_생명주기_v0.1]])
- **Output**
  - VPN/SSH 프로파일 구성
  - 원격 접속 세션 로그 (시작·종료·명령 녹화)
  - 접속 이력 월간 리포트

## 5. 수행 절차

### 5.1 사전 준비

1. 네트워크팀이 모든 운영 환경의 기본 방화벽 정책을 "원격 접속 차단"(deny-by-default) 으로 유지한다.
2. IAM 관리자가 원격 접속 인프라(VPN 게이트웨이·Bastion·제로트러스트 IAP) 와 MFA 연동을 운영한다.
3. 벤더 원격 지원 용도의 전용 계정·네트워크 구역(세그먼트) 을 사전 설계한다.

### 5.2 수행 단계

1. **신청**
   - 요청자가 [[TMP-ACC-004_원격접속_신청서_v0.1]] 를 작성한다. 필수: 사유·대상 시스템·작업 내용·소스 IP·요청 기간·업무 시간대.
   - 시스템 소유자가 업무 필요성을 확인하고 승인한다.
2. **심의**
   - 오동석 가 보안 영향을 검토한다. 운영 환경에 대한 특권 접속은 오동석 직접 승인, 일반 접속은 위임 승인 가능.
3. **접근 구성**
   - VPN 프로파일·Bastion 접근 권한을 발급한다.
   - 허용 소스 IP 화이트리스트를 방화벽에 추가한다.
   - 허용 시간대(예: 평일 09~18) 외 접속은 차단한다.
   - MFA 는 필수(VPN 로그인 및 내부 시스템 로그인 이중 적용).
4. **접속 및 세션 기록**
   - Bastion 은 모든 명령을 녹화한다(명령 로그·화면 녹화).
   - 운영 환경 대상 명령은 `sudo` 로그·SIEM 으로 전달한다.
   - 세션 최대 시간 4시간 초과 시 자동 종료하고 재인증을 요구한다.
5. **만료·회수**
   - 승인된 기간 만료 시 프로파일·화이트리스트가 자동 회수된다.
   - 작업 완료 즉시 본인이 조기 회수 요청할 수 있다.
6. **감사**
   - IAM 관리자가 월 1회 모든 원격 접속 세션 로그를 SIEM 대시보드로 검토한다.
   - 이상 명령(대량 파일 다운로드·계정 생성 등) 탐지 시 SecOps 에 전달한다.

### 5.3 완료 조건

- [ ] 기본 방화벽 정책이 deny-by-default 로 유지된다
- [ ] 모든 원격 접속이 [[TMP-ACC-004_원격접속_신청서_v0.1]] 에 근거한다
- [ ] VPN·Bastion 모두 MFA 가 강제된다
- [ ] 소스 IP 화이트리스트·허용 시간대가 적용된다
- [ ] Bastion 명령 녹화 및 SIEM 전달이 작동한다
- [ ] 승인 기간 만료 후 프로파일이 자동 회수된다

## 6. 인터페이스 부서

- **네트워크팀**: 방화벽·VPN 게이트웨이 운영
- **SecOps**: 세션 로그 분석·이상 탐지
- **계약·벤더 관리**: 외주 원격 지원 계약에 본 지침 준수 조항 반영

## 7. 주의사항 / 예외 처리

### 7.1 장애 긴급 원격 접속

심야·휴일 운영 환경 장애로 즉시 원격 접속이 필요한 경우, 대기 근무자(Oncall)가 사전 지정된 비상 원격 계정으로 접속하고 접속 후 1시간 이내 사후 신청서를 제출한다. 오동석 는 다음 영업일까지 사후 승인한다.

### 7.2 고정 IP 미보유 사용자

재택 근무자가 동적 IP 를 사용하는 경우, IP 화이트리스트 대신 MFA + 기기 인증서(Device Cert) + 지리적 위치 제한(국내 IP) 을 병행한다. 해외 출장 시 별도 신청으로 임시 허용한다.

### 7.3 MSP 현장 지원 원격 세션

병원(MSP)에 설치된 기기에 대한 원격 지원은 MSP 요청 기반으로만 개시한다. MSP 담당자 실시간 참관 + 세션 녹화 + 작업 종료 후 MSP 에 보고서 제출을 필수로 한다.

### 7.4 벤더 원격 지원 계약 해지

계약 종료·벤더 교체 시, IAM 관리자는 해당 벤더 계정·IP 화이트리스트·VPN 프로파일을 24시간 이내 회수하고 확인 증적을 남긴다. 회수 지연 시 오동석 에 보고.
