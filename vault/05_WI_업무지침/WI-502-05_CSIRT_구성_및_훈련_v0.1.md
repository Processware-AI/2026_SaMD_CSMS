# CSIRT 구성 및 훈련 업무지침 (WI-502-05)

> 상위 절차: [[PRO-MDCS-502_침해사고_대응_및_신고_절차_v1.0]] · §5 단계 13

## 1. 업무 목적

침해 대응에 즉시 가동 가능한 CSIRT(Computer Security Incident Response Team) 의 역할·책임·교육·훈련 체계를 수립·유지하여 인지부터 재발 방지까지 일관된 대응 역량을 보장한다.

## 2. 수행 주체

- **주 수행자**: CSIRT 리더
- **검토자**: CISO, 인사팀
- **승인자**: CEO

## 3. 범위

- CSIRT 구성원 선임·역할 정의
- 내부 전문가·외부 전문가 협력 체계
- 정기 교육·모의 훈련
- 훈련 결과의 대응 계획 개선 환류

## 4. 입력 자료 / 산출물

- **Input**
  - 조직 보안 인력 현황
  - 침해 사고 이력·교훈 학습
  - 외부 전문가 계약 목록
- **Output**
  - [[TMP-IR-006_CSIRT_역할_책임표_v0.1]]
  - [[TMP-IR-005_CSIRT_훈련_시나리오_v0.1]] 및 실행 기록
  - 훈련 결과 보고서·개선 과제

## 5. 수행 절차

### 5.1 사전 준비

1. CSIRT 리더가 핵심 역할을 정의한다: 리더·사건 관리자·포렌식 분석관·시스템 복구·커뮤니케이션 리드·법무·Safety.
2. 역할별 primary·backup 지정으로 24/7 커버리지 확보.
3. 외부 전문가(포렌식·법률) 와 SLA 계약 체결.

### 5.2 수행 단계

1. **CSIRT 구성**
   - 역할별 담당자를 [[TMP-IR-006_CSIRT_역할_책임표_v0.1]] 에 등록.
   - Primary/Backup 비상 연락망을 월간 갱신.
2. **교육**
   - 신규 구성원: 온보딩 교육(대응 절차·도구 사용·법규) 40시간 이내 완료.
   - 기존 구성원: 연 1회 이상 재교육.
3. **Tabletop 훈련**
   - 반기 1회 이상 시나리오 기반 토의형 훈련.
   - 시나리오 예시: 랜섬웨어·개인정보 유출·공급망 공격·AI 모델 공격.
4. **Simulation Exercise**
   - 연 1회 실제 환경 모의 훈련(Red Team vs CSIRT).
   - 외부 Red Team 활용 권장.
5. **훈련 기록·환류**
   - 각 훈련 후 [[TMP-IR-005_CSIRT_훈련_시나리오_v0.1]] 결과 섹션 작성.
   - 교훈 학습을 대응 계획·절차에 반영.
6. **숙련도 평가**
   - 반기 스킬 평가·역량 매트릭스 갱신.
   - 역량 Gap 은 외부 교육·자격증 취득으로 보완.

### 5.3 완료 조건

- [ ] CSIRT 역할·책임표가 완성되고 Primary/Backup 이 지정되었다
- [ ] 비상 연락망이 월간 갱신된다
- [ ] 신규 구성원 40시간 온보딩 교육이 완료된다
- [ ] 반기 Tabletop 1회·연간 Simulation 1회 이상 실시된다
- [ ] 훈련 결과가 대응 계획·절차 개선에 반영된다
- [ ] 외부 전문가(포렌식·법무) 와 SLA 계약이 유지된다

## 6. 인터페이스 부서

- **CISO**: 예산·외부 계약 승인
- **인사팀**: 교육 계획 등록·이수 관리
- **법무**: 외부 전문가 계약 검토
- **전 부서**: 훈련 참여(IT·개발·Safety·RA·홍보)

## 7. 주의사항 / 예외 처리

### 7.1 핵심 구성원 이탈

CSIRT 리더·포렌식 분석관 등 핵심 구성원이 퇴사·장기 이탈 시 Backup 이 즉시 승계하고, 3개월 이내 신규 인력을 충원한다. 공백 기간은 외부 전문가 계약으로 보강.

### 7.2 훈련 미참여

핵심 구성원이 훈련에 불참할 경우 해당 사유를 기록하고, 30일 이내 개별 훈련 또는 시나리오 리뷰로 대체 이수한다. 2회 연속 불참은 인사 평가에 반영.

### 7.3 실전 사건 중 훈련 일정

실제 침해 사건 대응 중에는 예정 훈련을 연기한다. 단 경미 사건은 실전을 훈련 사례로 변환하여 사후 리뷰 형태로 기록.

### 7.4 외부 기관 합동 훈련

KISA·ISAC·식약처 주관 합동 훈련 참여 요청 시 최우선 참여한다. 결과 공유 자료를 내부 훈련 라이브러리에 축적.
