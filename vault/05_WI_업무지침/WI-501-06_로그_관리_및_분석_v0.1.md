# 로그 관리 및 분석 업무지침 (WI-501-06)

> 상위 절차: [[PRO-MDCS-501_보안_모니터링_및_로그_관리_절차_v1.0]] · §5 단계 5~6

## 1. 업무 목적

보안 관련 활동·시스템 이벤트 로그를 **변조 방지 상태**로 보관하고 정기적으로 분석하여 침해 전 조기 경보와 침해 후 포렌식 증거를 확보하며, 규제 감사 시 증빙 가능성을 보장한다.

## 2. 수행 주체

- **주 수행자**: SecOps 엔지니어
- **검토자**: CISO, DPO(개인정보 관련 로그)
- **승인자**: CISO

## 3. 범위

- 애플리케이션 로그(인증·권한·의료 데이터 접근)
- 인프라 로그(OS·네트워크·DB·컨테이너)
- 보안 장비 로그(방화벽·IPS·AV·FIM)
- 의료기기 장치 로그(작동·알람·사용자 행위)

## 4. 입력 자료 / 산출물

- **Input**
  - 자산·서비스별 로그 소스 목록
  - 로그 필드 표준(ECS·CEF)
- **Output**
  - 로그 저장소(Hot·Warm·Cold) 구성
  - [[TMP-LOG-003_로그분석_보고서_v0.1]] 월간 작성물
  - 변조 방지 해시 체인 기록

## 5. 수행 절차

### 5.1 사전 준비

1. SecOps 가 활동 유형별 필수 로그 필드(누가·언제·무엇을·어디서·결과) 를 표준화한다.
2. 저장소 수명 정책을 정의한다: Hot 90일, Warm 1년, Cold 7년(침해 대응·규제 대비).
3. NTP 동기화와 변조 방지 저장소(Immutable) 구성 점검.

### 5.2 수행 단계

1. **로그 소스 등록**
   - 자산 등록 시 로그 수집 대상으로 함께 등록한다.
   - 누락 시 SIEM 대시보드에서 경고.
2. **수집·정규화**
   - 표준 필드로 정규화하고 필수 필드 누락 시 경보.
3. **변조 방지 보관**
   - append-only 인덱스·WORM 스토리지 또는 해시 체인으로 무결성 보장.
   - 주기적 해시 검증 작업을 스케줄링.
4. **접근 통제**
   - 로그 조회는 최소권한으로 허용, 개인정보 포함 로그는 DPO 승인.
   - 조회 자체를 감사 로그로 남긴다.
5. **정기 분석**
   - 월 1회 분석 보고서 작성: 상위 이상 이벤트·탐지율·False Positive 추이·SLA.
   - 분기 1회 침해 지표(IOC) 재평가.
6. **보존·파기**
   - 수명 정책에 따라 자동 티어링 및 파기.
   - 법정 홀드 대상 로그는 별도 격리.

### 5.3 완료 조건

- [ ] 모든 핵심 자산의 로그가 필수 필드를 포함하여 수집된다
- [ ] 변조 방지 저장소 또는 해시 체인이 운영된다
- [ ] 로그 접근 자체가 감사 로그로 기록된다
- [ ] 월간 분석 보고서가 작성·승인된다
- [ ] 보존 기간 정책이 자동 적용되고 법정 홀드가 관리된다
- [ ] 개인정보 포함 로그의 DPO 승인 체계가 작동한다

## 6. 인터페이스 부서

- **CSIRT**: 침해 발생 시 포렌식 증거 요청
- **DPO**: 개인정보 포함 로그 접근 심사
- **규제/RA**: 심사·감사 대응 증거 제출
- **인프라팀**: 로그 저장소 용량·성능 관리

## 7. 주의사항 / 예외 처리

### 7.1 저장소 용량 초과

Cold 저장소 용량 초과 시 SecOps 가 즉시 용량 확장 또는 보존 정책 재검토를 수행한다. 법정 보존 기간 이내의 로그는 결코 조기 삭제하지 않는다.

### 7.2 의료기기 오프라인 기간의 로그

네트워크 단절 상태에서 기기 내부에 누적된 로그는 재연결 시 일괄 동기화한다. 동기화 누락 위험을 줄이기 위해 기기 내부 버퍼 용량을 최소 7일로 설계.

### 7.3 로그 속 개인정보 최소화

과도한 개인정보가 평문으로 남는 경우 DPO 와 개발팀이 로그 필드 설계를 재검토하여 가명·마스킹을 적용한다. 변경 전 이미 적재된 로그는 사후 마스킹(일부 컬럼 재작성) 하거나 접근을 엄격 제한.

### 7.4 규제 감사 긴급 조회

식약처·개인정보보호위 등 규제기관 현장 조사 시 SecOps 팀장이 조회 권한을 한시 발급하고 조회 이력을 상세 기록한다. 조사 종료 후 즉시 권한 회수.
