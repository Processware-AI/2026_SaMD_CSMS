# 취약점 감시 및 접수 업무지침 (WI-601-01)

> 상위 절차: [[PRO-MDCS-601_취약점_감시_공개_조치_절차_v1.0]] · §5 단계 1

## 1. 업무 목적

시판 후 디지털의료기기의 취약점을 체계적으로 감시하여 악용 전에 식별하고, 내부 제보·외부 제보·자동 스캐닝으로 접수된 취약점을 단일 워크플로우로 통합 관리한다.

## 2. 수행 주체

- **주 수행자**: PSO, SBOM 관리자
- **검토자**: CISO, CSIRT
- **승인자**: CISO

## 3. 범위

- 내부 스캐닝(SAST/DAST/IAST·인프라 스캐너)
- SBOM-CVE 자동 연동
- 위협 인텔 피드·공개 공지
- 내부 임직원 제보

## 4. 입력 자료 / 산출물

- **Input**
  - CVE/NVD 피드·벤더 보안 공지
  - SBOM 최신본
  - 스캐너 리포트
- **Output**
  - [[TMP-VUL-001_취약점_접수_대장_v0.1]] (VUL ID 부여)
  - 주간 감시 요약 리포트

## 5. 수행 절차

### 5.1 사전 준비

1. PSO 가 감시 소스 목록을 유지한다: CVE/NVD·CISA KEV·벤더 피드·위협 인텔.
2. SBOM-CVE 자동 연동 도구(Dependency-Track 등) 를 운영한다.
3. 심각도 산정 방법(CVSS 4.0 + 임상 영향 가중치) 을 확정한다.

### 5.2 수행 단계

1. **자동 감시 (주 1회 이상)**
   - SBOM-CVE 매칭 잡을 주 1회 이상 실행.
   - 신규 매칭 발견 시 자동으로 취약점 대장에 티켓 생성.
2. **위협 인텔 감시**
   - PSO 가 CISA KEV·벤더 공지를 매일 확인한다.
   - 영향 가능성 항목은 접수 대장에 수동 등록.
3. **내부 제보 접수**
   - 임직원 전용 내부 포털에 제보 양식 제공.
   - 제보 즉시 VUL ID 부여 및 ACK 발송.
4. **분류**
   - CVSS 환산 심각도(Critical/High/Medium/Low) 와 영향 제품 범위 초기 판정.
5. **에스컬레이션**
   - Critical 취약점은 즉시 [[WI-601-03_식약처_MSP_통보_v0.1]] 로 이동.
   - 악용 정황이 있거나 악용 중(ITW) 인 경우 [[PRO-MDCS-502_침해사고_대응_및_신고_절차_v1.0]] IR 로 연계.
6. **주간 감시 리포트**
   - PSO 가 주 1회 감시 활동·신규 접수·진행 상태를 요약 배포.

### 5.3 완료 조건

- [ ] SBOM-CVE 자동 매칭이 주 1회 이상 실행된다
- [ ] CISA KEV·벤더 공지 일일 점검 루틴이 작동한다
- [ ] 내부 제보 ACK 가 24시간 이내 발송된다
- [ ] 접수 대장에 VUL ID·제보 경로·초기 심각도가 기록된다
- [ ] Critical 접수는 즉시 통보·IR 플로우에 연계된다
- [ ] 주간 감시 리포트가 정기 배포된다

## 6. 인터페이스 부서

- **SBOM 관리자**: CVE 매칭 자동화
- **CSIRT**: 악용 중 취약점 IR 연계
- **RA**: 규제 관련 공지 수집·영향 판단
- **개발팀**: 영향 제품 확정 협업

## 7. 주의사항 / 예외 처리

### 7.1 SBOM-CVE 도구 장애

자동 매칭 도구 장애 시 PSO 가 수동 매칭(영향 상위 20개 컴포넌트) 으로 대체하며, 복구 후 공백 기간을 재스캔한다.

### 7.2 0-day 공개 및 악용

0-day 공개 및 악용 중 사례는 감시 루틴을 기다리지 않고 즉시 트리아지 긴급팀을 구성한다. Critical 플로우로 직행하고, 임시 완화 조치(WAF·룰) 적용.

### 7.3 오픈소스 중단(Maintainer 포기)

사용 중인 오픈소스가 maintainer 포기로 패치 불가한 경우 교체 후보 평가·이관 계획을 수립한다. 단기적으로 우회 통제(네트워크 분리·권한 제한) 적용.

### 7.4 중복 제보

외부 VDP([[WI-601-02_VDP_운영_외부연구자_v0.1]]) 와 내부 제보가 중복되는 경우 단일 VUL ID 로 병합하고 관련자 모두에게 처리 상태를 동기화.
