# 전산실 강화보안 업무지침 (WI-501-08)

> 상위 절차: [[PRO-MDCS-501_보안_모니터링_및_로그_관리_절차_v1.0]] · §5 단계 0 (물리 보안구역·출입통제, 전산실 강화보안)

## 1. 업무 목적

전산실(서버실·데이터센터)에 대해 다단계 출입통제·랙 물리 잠금·케이블 관리·항온항습·전원 이중화·침입 감지 등 강화된 물리 보안 대책을 운영하여, 비인가 접근·장비 탈취·물리 훼손·환경 재해로 인한 디지털의료기기 운영 중단 및 데이터 유출을 방지한다.

## 2. 수행 주체

- **주 수행자**: 시설관리팀 (전산실 환경·출입 통제 운영)
- **검토자**: SecOps 팀장 (보안 통제 적합성 검토), 인프라팀 (장비 반출입·케이블 관리)
- **승인자**: CISO (출입 인가 목록 승인, 강화보안 정책 승인, 예외 승인)

## 3. 범위

- 자사 운영 전산실·서버실·네트워크 룸
- 코로케이션(Co-lo) 데이터센터 자사 전용 케이지·랙 구역
- 백업 미디어 보관실 및 테이프 라이브러리 구역
- 의료기기 현장 설치 서버 캐비닛(병원 내 랙 포함, 시설 협약 범위 내)

## 4. 입력 자료 / 산출물

- **Input**
  - 인가된 출입자 목록 (CISO 승인)
  - 장비 자산 목록 및 랙 배치도
  - [[TMP-FAC-010_전산실_출입_기록대장_v0.1]] (출입 이력)
  - [[TMP-FAC-011_전산실_환경_점검_일지_v0.1]] (온도·습도·UPS·소화 설비 점검)
  - [[TMP-FAC-012_장비_반출입_신청서_v0.1]] (장비 반입·반출 요청)
- **Output**
  - 완료된 [[TMP-FAC-010_전산실_출입_기록대장_v0.1]] (월별 집계 포함)
  - 완료된 [[TMP-FAC-011_전산실_환경_점검_일지_v0.1]] (주간·월간)
  - 승인된 [[TMP-FAC-012_장비_반출입_신청서_v0.1]]
  - 이상 탐지 보고서 (환경 이상·비인가 출입 시도)
  - 분기 물리 보안 점검 보고서

## 5. 수행 절차

### 5.1 사전 준비

1. 시설관리팀은 전산실 출입 인가자 목록을 분기마다 CISO 승인을 받아 최신화하고, 퇴사·직무 변경 시 즉시 갱신한다.
2. 출입통제 시스템(카드키·생체인식 등)의 출입 인가 설정이 CISO 승인 목록과 일치하는지 월 1회 검증한다.
3. 전산실 내 CCTV 카메라가 모든 출입구·서버 랙 열·중요 장비를 커버하고 있는지 확인하고, CCTV 영상을 최소 90일 이상 보관한다.

### 5.2 수행 단계

1. **다단계 출입통제 운영**
   - 전산실 출입은 최소 2단계 인증(카드키 + 생체인식 또는 PIN)을 요구한다.
   - 출입 시 성명·소속·목적·시간이 [[TMP-FAC-010_전산실_출입_기록대장_v0.1]]에 자동(카드 리더) 또는 수기로 기록된다.
   - 방문자·협력업체는 반드시 시설관리팀 직원 또는 인프라팀 직원이 동행 에스코트 하에 입실하며, 단독 출입을 금지한다.
   - 퇴실 시 모든 반출 장비가 [[TMP-FAC-012_장비_반출입_신청서_v0.1]]와 일치하는지 확인한다.

2. **랙 물리 잠금 및 케이블 관리**
   - 모든 서버 랙에는 전면·후면 잠금 가능한 도어를 장착하고, 업무 외 시간에는 잠금 상태를 유지한다.
   - 랙 잠금 키는 시설관리팀이 관리하며, 키 사용 이력을 기록한다. 마스터 키 사본은 금고에 보관하고 CISO 승인 아래 접근한다.
   - 케이블(전원·데이터·관리 콘솔)은 색상 코딩·레이블링하여 유형을 식별 가능하게 한다.
   - 미사용 랙 공간에는 블랭킹 패널을 설치하여 케이블 훼손 및 공기 단락을 방지한다.
   - 전원 케이블과 데이터 케이블은 별도 케이블 트레이로 분리 배선하여 전자기 간섭을 최소화한다.

3. **환경 모니터링 및 이상 대응**
   - 항온항습 장치(CRAC/CRAH)의 온도(권장 18~27°C)·습도(권장 40~60%)를 환경 모니터링 시스템으로 24시간 연속 감시한다.
   - 온도가 27°C 초과 또는 습도가 60% 초과 시, 환경 모니터링 시스템이 담당자에게 자동 알림을 발송한다.
   - 경보 발생 시 시설관리팀이 15분 이내 현장 확인하고, 2시간 이내 정상 복구가 불가한 경우 인프라팀에 장비 셧다운 필요성을 통보한다.
   - 주간 점검 및 월간 정밀 점검 결과를 [[TMP-FAC-011_전산실_환경_점검_일지_v0.1]]에 기록한다.

4. **전원 이중화 및 UPS 관리**
   - 주 전원 이중화(이중 PDU, A/B 전원 공급)가 유지되고 있는지 월 1회 점검한다.
   - UPS 배터리 상태를 매월 점검하고, 배터리 수명(제조사 기준)이 80% 이하로 감소하면 교체 계획을 수립한다.
   - 전원 절체 테스트(UPS → 발전기 또는 주 전원 복구)를 반기 1회 이상 실시하고 결과를 기록한다.
   - 전원 관련 이상 경보 발생 시 즉시 시설관리팀장 및 인프라팀장에게 통보한다.

5. **소화 설비 및 침입 감지 점검**
   - 가스 소화 시스템(청정 소화약제)의 압력·노즐 상태를 월 1회 육안 점검하고, 연 1회 전문업체 법정 점검을 실시한다.
   - 연기 감지기·열 감지기 정상 작동 여부를 분기 1회 테스트한다.
   - 전산실 문·창문에 설치된 침입 감지 센서(적외선·자기 접점)의 동작 상태를 월 1회 점검한다.
   - 침입 경보 발생 시 시설관리팀이 즉시 현장 확인하고, 비인가 인원 발견 시 보안팀 및 CISO에 즉시 보고한다.

6. **장비 반출입 관리**
   - 모든 장비 반입·반출은 사전에 [[TMP-FAC-012_장비_반출입_신청서_v0.1]]를 작성하고 인프라팀장 및 시설관리팀장의 승인을 받는다.
   - 반출 장비에는 자산 태그가 부착되어 있는지 확인하고, 반출 후 자산 목록을 업데이트한다.
   - 폐기 장비의 경우, 데이터 완전 삭제(DoD 기준 이상)를 인프라팀이 확인한 후 반출한다.

7. **분기 종합 점검 및 보고**
   - 시설관리팀장은 분기마다 출입 이력 분석·환경 이상 이력·장비 반출입 현황·CCTV 운영 상태를 종합하여 물리 보안 점검 보고서를 작성하고 CISO에 제출한다.
   - 비인가 출입 시도, 환경 임계값 초과, 소화·침입 감지 오작동 사례를 분석하여 개선 사항을 도출한다.

### 5.3 완료 조건

- [ ] 출입 인가자 목록이 분기 최신화되고 CISO 승인이 확인된다
- [ ] [[TMP-FAC-010_전산실_출입_기록대장_v0.1]]에 전 출입 이력이 빠짐없이 기록되어 있다
- [ ] 모든 서버 랙의 전면·후면 도어 잠금 상태가 확인된다
- [ ] 주간 환경 점검 결과가 [[TMP-FAC-011_전산실_환경_점검_일지_v0.1]]에 기록되어 있다
- [ ] UPS 배터리 상태 점검이 완료되었다
- [ ] 장비 반출입 전수에 대해 [[TMP-FAC-012_장비_반출입_신청서_v0.1]] 승인 기록이 존재한다
- [ ] 분기 종합 물리 보안 점검 보고서가 CISO에 제출되었다

## 6. 인터페이스 부서

- **인프라팀**: 랙 배치도·장비 자산 목록 관리, 장비 반출입 신청 및 데이터 삭제 확인
- **SecOps 팀**: 물리 보안 통제 적합성 검토, CCTV 영상 접근 권한 관리
- **CISO**: 출입 인가 목록 승인, 강화보안 정책 승인, 분기 보고서 수신
- **외부 용역업체(소화·전기·공조)**: 법정 점검 및 유지 보수 수행 (에스코트 필수)

## 7. 주의사항 / 예외 처리

### 7.1 긴급 출입 처리

장애 대응 등으로 인가받지 않은 인원의 긴급 출입이 필요한 경우, CISO 또는 위임받은 인프라팀장의 구두 승인 후 시설관리팀이 동행 에스코트 하에 임시 출입을 허용한다. 긴급 출입 후 24시간 이내 [[TMP-FAC-010_전산실_출입_기록대장_v0.1]]에 사유와 구두 승인자를 기재하고 정식 출입 인가 절차를 사후 진행한다.

### 7.2 전산실 환경 비상 상황(과열·침수·화재)

환경 임계값(온도 30°C 초과, 침수 감지, 화재 경보)이 발생한 경우:
- 시설관리팀이 즉시 인프라팀에 장비 긴급 셧다운 여부 판단을 요청한다.
- 화재 경보 시 가스 소화 시스템이 자동 작동하기 전 전산실 내 인원 퇴피를 최우선으로 확인한다.
- 침수 발생 시 전원 차단을 1순위로 조치하고, PRO-MDCS-502(침해사고 대응) 또는 BCP를 트리거한다.
- 환경 비상 상황의 원인·피해·복구 내역을 전산실 환경 사고 보고서로 작성하여 CISO에 제출한다.

### 7.3 코로케이션 데이터센터 관리

외부 코로케이션 시설 이용 시, 데이터센터 사업자의 물리 보안 기준이 SaMD-CSMS 물리 보안 요건에 부합하는지 계약 전 확인하고, 연간 SOC 2 보고서 또는 ISO 27001 인증서를 수령하여 보관한다. 자사 케이지·랙은 별도 잠금 장치를 추가하고, 반출입 이력은 자사 [[TMP-FAC-012_장비_반출입_신청서_v0.1]]로 관리한다.
