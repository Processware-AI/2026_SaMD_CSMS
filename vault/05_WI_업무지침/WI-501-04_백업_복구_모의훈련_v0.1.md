# 백업 및 복구 모의훈련 업무지침 (WI-501-04)

> 상위 절차: [[PRO-MDCS-501_보안_모니터링_및_로그_관리_절차_v1.0]] · §5 단계 8~9

## 1. 업무 목적

랜섬웨어·저장소 손상·운영 실수 등으로 인한 데이터 손실 시 신속한 복구가 가능하도록 정기 백업과 반기 복구 모의훈련을 운영하여 RTO·RPO 목표 달성 능력을 검증한다.

## 2. 수행 주체

- **주 수행자**: 인프라팀 엔지니어, SecOps
- **검토자**: PSO, 개발팀
- **승인자**: 오동석

## 3. 범위

- 소스코드 리포지토리 (Git 미러)
- 운영 데이터베이스·파일 저장소
- 구성 관리(IaC) 리포지토리
- 암호키·인증서 (§7.1 예외)

## 4. 입력 자료 / 산출물

- **Input**
  - 자산·서비스별 RTO/RPO 목표
  - 백업 정책·보존 기간·오프사이트 위치
- **Output**
  - 백업 실행 로그·성공률
  - [[TMP-LOG-002_복구_모의훈련_결과서_v0.1]]
  - 훈련 기반 개선 이슈 리스트

## 5. 수행 절차

### 5.1 사전 준비

1. 자산별 RTO(목표 복구 시간) 와 RPO(목표 복구 시점) 를 확정한다. 핵심 서비스는 RTO ≤ 4시간, RPO ≤ 15분.
2. 3-2-1 백업 원칙(3개 사본·2개 매체·1개 오프사이트) 을 적용한다.
3. 백업 암호화·무결성 해시 정책을 정의한다.

### 5.2 수행 단계

1. **일일 백업**
   - 자동 스케줄로 증분 백업을 수행한다.
   - 실패 알림을 SIEM 에 전달하고 1시간 이내 재수행 또는 티켓 발행.
2. **주간 전체 백업**
   - 주 1회 전체 백업 후 오프사이트 저장소에 복제한다.
   - 해시·디지털 서명으로 무결성을 보장한다.
3. **보존·파기**
   - 보존 기간(운영 30일·아카이브 7년) 에 따라 자동 파기한다.
   - 파기 로그를 [[WI-101-03_증적_수집_기준_v0.1]] 기준으로 보관.
4. **복구 모의훈련**
   - 반기 1회 이상 핵심 서비스 대상으로 실제 복구 훈련을 수행한다.
   - 시나리오: 랜섬웨어, 저장소 전체 손실, 구성 변경 실수.
   - 목표 RTO/RPO 준수 여부, 데이터 무결성 검증 결과를 기록한다.
5. **결과 보고·개선**
   - [[TMP-LOG-002_복구_모의훈련_결과서_v0.1]] 작성.
   - 식별된 개선 이슈는 티켓으로 발행하고 다음 훈련 전까지 완료.
6. **랜섬웨어 특화 대비**
   - 오프라인·Air-gap 백업 1세트 유지.
   - 백업 저장소의 쓰기 전용 역할(WORM) 구성.

### 5.3 완료 조건

- [ ] 자산별 RTO/RPO 목표가 문서화되었다
- [ ] 3-2-1 백업 원칙이 운영된다
- [ ] 일일·주간 백업 성공률이 99.5% 이상이다
- [ ] 반기 복구 모의훈련이 실시되고 결과서가 작성되었다
- [ ] 식별된 개선 이슈가 티켓으로 관리된다
- [ ] 랜섬웨어 대비 오프라인 백업이 유지된다

## 6. 인터페이스 부서

- **개발팀**: 데이터베이스 정합성 검증 참여
- **SecOps**: 백업 무결성·접근 로그 감사
- **CSIRT**: 침해 대응 중 복구 수행
- **시설관리팀**: 오프사이트 보관 물리 보안

## 7. 주의사항 / 예외 처리

### 7.1 암호키 백업 분리

암호키는 본 WI 범위에서 제외하며 [[WI-402-04_암호키_수명주기_v0.1]] 의 N-of-M 분할 백업으로 별도 관리한다. 일반 백업과 혼재 금지.

### 7.2 개인정보 포함 백업

개인정보 포함 백업은 암호화 필수. 파기 시 암호화 키 폐기로 데이터 소거(Crypto-erase) 를 적용할 수 있다. DPO 승인 필요.

### 7.3 대규모 장애 실전 복구

복구 모의훈련이 아닌 실제 장애 상황에서는 오동석 주관 IR Command Center 에서 결정권을 통합하며, 훈련 절차를 그대로 실행한다. 복구 후 사후 리뷰에서 훈련 유효성을 검증한다.

### 7.4 클라우드 리전 장애

단일 클라우드 리전 전체 장애 대비 멀티 리전 DR 구성을 유지한다. 주기적 DR 페일오버 훈련을 수행하고 결과를 연간 BCP 보고서에 반영.
