# 물리 보안구역 및 출입통제 업무지침 (WI-501-07)

> 상위 절차: [[PRO-MDCS-501_보안_모니터링_및_로그_관리_절차_v1.0]] · §5 단계 0

## 1. 업무 목적

디지털의료기기 개발·운영 시설의 물리적 위협(침입·화재·침수·정전 등) 을 예방하고, 인가된 인원에게만 출입을 허용하여 물리적 경로를 통한 사이버 공격을 차단한다.

## 2. 수행 주체

- **주 수행자**: 시설관리팀 담당자
- **검토자**: 오동석, 네트워크팀(CCTV·NAC 연동)
- **승인자**: 오동석

## 3. 범위

- 사무실·개발실·품질 검사실
- 서버실·전산실·KMS 룸(강화구역)
- 자산 보관 창고·백업 저장소
- 주요 장비(의료기기 테스트 벤치) 보관소

## 4. 입력 자료 / 산출물

- **Input**
  - 시설 평면도·구역 분류
  - 인가자 목록·방문자 관리 정책
- **Output**
  - 구역별 출입 권한 매트릭스
  - [[TMP-PHY-001_출입기록_대장_v0.1]]
  - CCTV 녹화 보존 기록

## 5. 수행 절차

### 5.1 사전 준비

1. 시설관리팀이 시설을 등급으로 분류한다: 공개·일반·제한·강화.
2. 각 구역의 물리 보안 설비(CCTV·UPS·소화·침입감지·침수감지) 구축 상태를 확인한다.
3. 출입 카드 시스템과 방문자 관리 시스템을 구축한다.

### 5.2 수행 단계

1. **구역 지정·표시**
   - 구역 등급을 출입문에 표지로 명확히 한다.
   - 강화구역(전산실 등) 은 이중 잠금·다단계 인증.
2. **출입 권한 부여**
   - 직무·업무 기반 최소권한 원칙으로 카드 권한을 부여한다.
   - 강화구역은 오동석 승인 필요.
3. **방문자 관리**
   - 방문자는 사전 신청·신분증 확인·임시 카드 발급·동반자 필수.
   - 출입기록 대장에 방문 목적·시간·동반자를 기록.
4. **CCTV·침입감지**
   - 주요 구역은 24시간 녹화, 보존 기간 최소 30일.
   - 움직임·문 열림 이벤트는 침입감지 시스템에서 알림.
5. **UPS·소화·침수 설비**
   - 전산실은 UPS + 발전기 백업, 분기별 점검 수행.
   - 소화 설비(가스식)·누수 감지기를 유지한다.
6. **주기 검토**
   - 반기 1회 오동석 주관 출입 권한 검토 수행.
   - 연 1회 시설 보안 감사(외부 점검 포함).

### 5.3 완료 조건

- [ ] 시설 구역이 등급별로 분류·표시되었다
- [ ] 강화구역에 이중 잠금·다단계 인증이 적용된다
- [ ] 출입 권한이 직무 기반 최소권한으로 부여되었다
- [ ] 방문자 출입기록 대장이 일관되게 유지된다
- [ ] CCTV 녹화 보존(≥30일) 과 침입감지가 작동한다
- [ ] UPS·소화·침수 설비 점검 이력이 유지된다
- [ ] 반기 출입 권한 검토가 완료되었다

## 6. 인터페이스 부서

- **네트워크팀**: 출입통제 시스템 네트워크 구성
- **HR**: 임직원 입·퇴사 이벤트 연동
- **법무**: 개인정보(CCTV 영상) 처리 자문
- **CSIRT**: 물리 침입 시 IR 연계

## 7. 주의사항 / 예외 처리

### 7.1 강화구역 응급 출입

전산실·KMS 룸 응급 상황에서 오동석 승인 대기 없이 Oncall 관리자가 출입이 필요한 경우, 2인 1조·사전 지정된 비상 연락망 통보 후 입실한다. 입실 직후 사유·영상 증빙을 오동석 에 보고.

### 7.2 외부 업체 유지보수 작업

벤더 엔지니어 방문은 계약서의 비밀 유지·보안 준수 조항을 사전 확인한다. 작업 시간 내내 자사 담당자 동반 필수.

### 7.3 CCTV 영상 요청

수사기관·소송 관련 영상 요청은 법무 검토 후 공식 영장/요청서 확인하여 제공한다. 요청·제공 이력을 대장에 기록.

### 7.4 시설 침수·화재 발생

침수·화재 감지 시 즉시 대피·소화 조치와 병행해 서버 자동 셧다운·데이터 백업 상태를 확인한다. 사고 후 [[PRO-MDCS-502_침해사고_대응_및_신고_절차_v1.0]] 와 별개로 BCP 를 발동한다.
