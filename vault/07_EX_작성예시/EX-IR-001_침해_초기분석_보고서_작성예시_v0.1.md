# 침해 초기 분석 보고서 작성예시 (EX-IR-001)

| IR ID | IR-2026-017 ← 연도별 순차 번호 |

| 항목 | 내용 |

| 침해 유형 | 랜섬웨어 감염 의심 ← 확정 전 "의심" 표기 |
| 영향 자산 | fs-radiology-01 (영상 파일 서버), 웹 PACS 프런트(1대) |
| 영향 MSP(해당 시) | S병원(파일럿 MSP) — 동일 네트워크 미설치, 영향 없음 추정 |

## 작성 요령

- 타임스탬프는 KST + 타임존 병기.
- "확정" 과 "의심" 을 명확히 구분. 불확실은 반드시 "미확인·조사 중" 표기.
- 증거 해시는 전체 값을 기록 (예시에서는 공간상 생략 — 실제는 64자 전체).
- 보고 일시(작성 시각)는 인지 일시 기준 **2시간 이내** 작성을 원칙으로 하며, 초과 시 사유를 기본 정보 비고란에 기재한다. 인지~보고 경과 시간이 SLA 를 초과하면 해당 건을 [[WI-502-01_침해사고_대응_초기조치_v0.1]] 개선 이슈로 이관한다.
