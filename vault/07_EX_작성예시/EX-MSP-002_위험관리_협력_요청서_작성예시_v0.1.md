# 위험관리 협력 요청서 작성예시 (EX-MSP-002)

> 원본 양식: [[TMP-MSP-002_위험관리_협력_요청서_v0.1]]

> **이 문서는 교육용 예시로, 실제 기록(REC)이 아닙니다.**

## 1. 작성 개요

서울메디컬센터에서 운영 중인 AI 기반 심전도 분석 SaMD에서 인증 우회 취약점(CVE-2026-10234)이 공개 보안 권고를 통해 확인되었다. 해당 취약점은 원격 공격자가 인증 없이 분석 결과에 접근할 수 있어 환자 정보 유출 및 오진 위험이 존재한다. 서울메디컬센터 의료정보보안팀은 제조업자등과 협력하여 위험 관리 활동을 수행하기 위해 본 요청서를 제출한다.

## 작성 요령

1. CVE 번호가 있는 경우 반드시 기재하고, NVD/벤더 권고 원문을 첨부한다. 제조업자등의 분석 속도를 높이는 핵심 정보이다.
2. MSP 측 임시 대응 조치는 구체적 행동 단위로 번호를 매겨 기재한다. "조치 중"과 같은 모호한 표현을 피한다.
3. 협력 요청 사항의 희망 일정은 영업일 기준으로 명시하며, 환자 안전 영향도에 비례하여 긴급도를 반영한다.
4. 위험 평가는 MSP 측 판단임을 명시하고, 최종 위험 등급은 제조업자등과 협의하여 결정한다.
