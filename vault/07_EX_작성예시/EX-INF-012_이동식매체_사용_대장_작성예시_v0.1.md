## 작성 요령

- **이동식매체 사용 현황**: 불출 시 즉시 기재합니다. 반납일과 데이터 삭제 확인은 반납 당일 기재하며, 미반납 상태는 "미반납"으로 표시합니다.
- **데이터 삭제 확인**: 반납 시 담당자가 직접 삭제 여부를 확인한 후 "확인"으로 기재합니다. 기밀 데이터가 포함된 경우 삭제 방법도 비고에 기재하세요.
- **분실·도난 현황**: 해당 없는 경우 "해당 없음"으로 기재합니다. 분실·도난 발생 시 즉시 보안팀장과 오동석에게 보고하고, 조치 내용을 구체적으로 기재하세요.
- **비고**: 현재 사용 중인 매체가 있는 경우 반드시 비고에 기재합니다. 허용 기간을 초과한 미반납 매체는 담당자가 사용자에게 즉시 반납을 요청해야 합니다.
