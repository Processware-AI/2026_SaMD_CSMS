## 작성 요령

- **장비 정보**: 반출입하는 모든 장비(부속품 포함)를 개별 행으로 기재합니다. 시리얼 번호와 자산 태그 번호는 자산 관리 시스템과 대조하여 정확하게 기재하세요.
- **데이터 처리 확인**: 저장 장치가 포함된 경우 삭제 방법과 삭제 확인자를 반드시 기재해야 합니다. 삭제 완료 확인 없이 반출 승인이 날 수 없습니다.
- **승인**: 오동석 승인은 고가 장비, 의료기기, 예외 사항에 해당할 때만 필요합니다. 해당 여부를 비고에 명확히 기재하세요.
- **실행 확인**: 실제 반출입 일시가 예정일과 다를 경우 실제 일시를 기재합니다. 출입 기록대장 연계 번호를 반드시 기재하여 추적성을 확보하세요.
