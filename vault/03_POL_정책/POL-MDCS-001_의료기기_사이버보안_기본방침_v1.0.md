# 의료기기 사이버보안 기본방침 (POL-MDCS-001)

> 상위 기준: [[표준프로세스_구성원칙]] · 문서체계: [[01_문서체계]] · 표준 개요: [[00_SaMD-CSMS_표준개요]]

## 1. 목적

본 방침은 「디지털의료제품법」 제14조(사이버보안 업무의 수행·문서화) 및 제32조(취약점 감시·대응)의 이행을 위해, 당사가 공급하는 디지털의료기기를 **설계부터 폐기까지(Lifecycle)** 전자적 침해행위로부터 보호하기 위한 **최상위 경영 의지와 방향**을 명시한다.

## 2. 적용 범위

- 당사가 개발·제조·유통·유지보수하는 모든 디지털의료기기 (AI 적용 기기, 소프트웨어 의료기기, 임베디드 기기 포함)
- 디지털의료기기의 액세서리·전자 인터페이스·구성품
- 디지털의료기기의 **제품 수명주기 전반** — 설계·개발·검증·배포·운영·유지·수명 종료(EOS)·레거시·폐기
- 당사 직원 및 위·수탁 개발·운영 파트너
- 의료서비스제공자(MSP)와의 협력·통보 활동은 선택 실행 지침 범위에서 적용

## 3. 정책 원칙

1. **Security by Design & Default** — 위협 모델링과 보안 요구사항을 개발 초기 단계에 내재화하며, 출하 시 기본 설정이 안전한 상태가 되도록 설계한다.
2. **위험 기반 관리(Risk-based Management)** — 환자 안전·임상 영향을 최우선으로 고려하여 위협·취약점을 식별·평가하고, 잔여 위험의 수용 여부를 경영진이 판단한다.
3. **수명주기 책임(Lifecycle Accountability)** — 출시부터 지원 종료(EOS) 이후까지 보안 조치 책임을 명확히 하며, 수명 종료 시점 및 이후 지원 계획을 사전에 공지한다.
4. **투명한 대응과 협력(Transparency & Coordination)** — 침해행위·취약점 발견 시 식품의약품안전처·의료서비스제공자·이용자에게 지체 없이 알리고, 정보 공유와 공동 대응을 기본 원칙으로 한다.
5. **문서화와 증적(Documented Evidence)** — 모든 보안 활동은 그 수행 사실·절차·책임·도구·결과물을 문서화하여 심사·조사·규제 대응 시 재현 가능하도록 유지한다.

## 4. 역할과 책임

| 역할 | 책임 |
|---|---|
| **CEO** | 본 방침 최종 승인, 사이버보안 자원 배정, 중대 침해행위 대외 의사결정 |
| **CISO (Chief Information Security Officer)** | 본 방침 유지·연 1회 유효성 검토, 전사 사이버보안 전략 수립, 침해대응 총괄 |
| **Product Security Council (PSC)** | 제품 보안 거버넌스 심의체. 리스크 수용·예외 승인, 보안 KPI 리뷰 (분기) |
| **Product Security Officer (PSO) / 제품 보안 책임자** | 제품별 보안 활동 총괄, 위협 모델링·보안 SDLC·SBOM·침해대응·취약점 관리 전반의 실행 책임 |
| **R&D / 개발팀** | 보안 요구사항 반영 설계·보안 코딩·보안 테스트 수행 |
| **QA / 검증팀** | 보안 테스트(SAST/DAST/Fuzzing/Pentest) 유효성 확인·결과 문서화 |
| **SecOps / 보안운영팀** | 운영 환경 모니터링·로그 분석·안티바이러스·백업·침해 탐지 |
| **CSIRT (Computer Security Incident Response Team)** | 침해행위 탐지·분석·격리·복구·사후분석, 24시간 비상 연락 유지 |
| **Regulatory Affairs (RA)** | 식품의약품안전처 신고·통보·인허가 대응, 법규 모니터링 |
| **모든 임직원** | 본 방침·하위 절차 준수, 보안 이상 징후 즉시 보고 |

## 5. 준수 기준

- 제3~22조의 모든 보안 활동은 문서화하여 보관한다 ([[PRO-MDCS-101_보안_문서화_관리_절차_v1.0]]).
- 제품 수명주기 전반에 걸친 위협 모델링·위험 평가·잔여 위험 재평가를 주기적으로 수행한다 ([[PRO-MDCS-201_사이버보안_리스크_관리_절차_v1.0]]).
- 침해행위 인지 즉시 식품의약품안전처장·의료서비스제공자·이용자에게 지체 없이 알린다 (24시간 비상 연락 체계 유지).
- 본 방침은 연 1회 이상 유효성을 검토하며, 법규·산업 동향·주요 침해사건을 반영하여 개정한다.
- 권고(선택 실행) 지침은 원칙적으로 준수하는 것을 기본으로 하되, 비용·편익·임상 영향을 고려하여 PSC 의 판단으로 예외를 명시적으로 결정한다.
